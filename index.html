About From the Cupboard

What this is

From the Cupboard is a recipe idea generator. You select the ingredients you have at home, optionally set a flavor mood or cuisine preference, and the app suggests recipe ideas. You can then get a full recipe generated on the spot, or search for human-made versions online.

How it works

The ingredient selection and all interface logic runs directly in your browser — no account or tracking. When you ask for recipe suggestions, your ingredient list is sent to a language model (currently a large open-source LLM via the Groq API), which generates ideas and recipes based strictly on what you have.

The backend is a small Cloudflare Worker that acts as a proxy, so no API key ever appears in the page source. Everything else — the UI, ingredient data, and state management — is plain HTML, CSS, and JavaScript, hosted on GitHub Pages.

Who made it

This is a personal project by Tom Offrede. It was built as an experiment in vibe-coding and from a desire to have a sous-chef in my kitchen.

Source code: github.com/tomoffrede/FromTheCupboard

Sobre o From the Cupboard

O que é?

From the Cupboard é um gerador de ideias de receitas. Você seleciona os ingredientes que tem em casa, define opcionalmente um sabor ou culinária preferida, e o app sugere ideias de receitas. Depois, você pode gerar uma receita completa na hora ou buscar versões criadas por humanos online.

Como funciona?

A seleção de ingredientes e toda a lógica de interface rodam diretamente no seu navegador — sem conta nem rastreamento. Quando você pede sugestões de receitas, sua lista de ingredientes é enviada a um modelo de linguagem (atualmente um grande LLM de código aberto via API Groq), que gera ideias e receitas estritamente com base no que você tem.

O backend é um pequeno Cloudflare Worker que age como proxy, de modo que nenhuma chave de API aparece no código-fonte da página. Todo o resto — UI, dados de ingredientes e gestão de estado — é HTML, CSS e JavaScript puro, hospedado no GitHub Pages.

Quem fez?

Este é um projeto pessoal de Tom Offrede. Foi criado como um experimento de vibe-coding e do desejo de ter um sous-chef em casa.

Código-fonte: github.com/tomoffrede/FromTheCupboard
